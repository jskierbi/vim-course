### LESS

* `--follow`, `-f` or inside less, **F** character: follow

* **/** search forward
* **?** search backwards
* **n** next occurrence
* **N** prev occurrence
* **G** go to the end of file
* **g** go to the start of the file
* **ctrl+f, ctrl+b** page forward/backward
* **ma**, **'a** mark position with marker ***a***, go to marker ***a***
